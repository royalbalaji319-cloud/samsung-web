# Samsung Website Clone (HTML & CSS)

This project is a **Samsung-style website clone** built using **pure HTML and CSS**.  
It is designed to look clean, modern, and professional, similar to the official Samsung website.

---

##  Features

- Clean and modern UI design  
- Built using only **HTML5** and **CSS3**  
- Responsive layout (Desktop & Mobile friendly)  
- Separate HTML and CSS files  
- Product sections for Mobiles, Laptops, TVs, etc.  
- Beginner-friendly frontend project  

---

##  Technologies Used

- HTML5  
- CSS3  
- Visual Studio Code
